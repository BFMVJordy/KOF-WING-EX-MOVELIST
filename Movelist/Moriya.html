MORIYA

★ MODO OSCURO

Guía de movimientos e instrucciones en español

Sistema de Espada: Moriya tiene dos posturas de guardia (居合/Iaido): Guardia ₁ (↓ ← + A/C) activa la postura agachada con espada envainada, desde la que se pueden ejecutar los ataques Neblinoso (alto, medio, bajo). Guardia ₂ (↓ ← + B/D) activa la postura de pie, desde la que se puede avanzar (Paso Lunar) o contraatacar (Iaido Lunar). Presionar L durante cualquier Guardia cancela y desenvaina la espada.

Poder Divino · Guardia Iaido
Escondido		 Cualquier → ← ↑ ↓ + O — Movimiento muy complejo con múltiples fases:
Fase 1 (agachado, espada envainada): Se puede cargar. Si el usuario tiene 1 barra de energía o más y recibe un ataque, gasta 1 barra automáticamente para bloquear y contraatacar. También se puede soltar O manualmente para atacar sin gastar energía. Si el ataque impacta al rival puede derivar en el siguiente nivel.
Fase 2 (de pie, espada envainada cargando): Consume rápidamente toda la energía restante. Durante esta fase se puede usar adelante/atrás + esquive para moverse instantáneamente con invencibilidad.
Fase 3 (ataque final): Al soltar O lanza un ataque no bloqueable. Cada barra de energía consumida aumenta el daño: menos de 1 barra = 25%, 1 barra = 50%, 2 barras = +60%, 3 barras = +70%. Tras el 3.er golpe se puede derivar presionando ↓ → + A/C para un ataque adicional. Las primeras dos fases pueden ser esquivadas por el rival.
